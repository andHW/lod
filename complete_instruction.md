# Full Instruction

This is the full instruction to manipulate the weaving machine (Loom of Doom).
Please note that this instruction is not intended to be read before exploring
the weaving device on your own.

## Basic Key Mappings

| key     | function      |
| ------- | ------------- |
| `W`     | Up            |
| `A`     | Down          |
| `S`     | Right         |
| `D`     | Left          |
| `H`     | Left Shift    |
| `L`     | Right Shift   |
| `Space` | Punch All     |
| `0-9`   | swap cards    |
| `B`     | remove border |
